# Frontend (Next.js)

Dashboard for Price Pulse Pro price history and API status.

- **Dev:** `npm run dev` (port from `FRONTEND_PORT` in `.env`, default 3100)
- **API:** reads `NEXT_PUBLIC_API_BASE_URL` (see root `.env.example`)
- **Tests:** `npm test` from this directory or `make test` at root.
